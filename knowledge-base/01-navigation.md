## Dashboard Navigation — Complete Guide

### Logging In
1. Go to **hub.veent.io**
2. Click **Login** and sign in with your Google account
3. You land on the **Dashboard Home** page

### Dashboard Home — What You See
- **Top header bar**: Veent logo, "Dashboard" link, "Wallet" link, your name (top right)
- **Welcome message**: "Welcome back, [Your Name]"
- **My Events** table with all your events
- **Create Event** button (top right)

### My Events Table Columns
| Column | What It Shows |
|--------|--------------|
| **Event** | Event name, location, "Open Link" button |
| **Event Schedule** | Start and end dates/times |
| **Created At** | When event was created |
| **Status** | Current status (Live, Unlisted, Draft, Ended, Archived) |
| **Payment** | Payment mode (Test or Live) |
| **Created By** | Who created the event |
| **Actions** | Three-dot menu (⋮) with options: Open Link, Scanner, Copy Link, Duplicate, Share |

### Opening an Event
Click on any event name in the table. This takes you inside that event's dashboard.

### Inside an Event — 7 Tabs
Once inside an event, you see:
- **Event header**: Event name, dates, location, public URL, "Edit Event" button (red), status badge
- **7 horizontal tabs**: Click any to switch sections

| Tab | What's Inside | Key Buttons |
|-----|--------------|-------------|
| **Analytics** | Sales charts, revenue, ticket stats | View Breakdown, date filters |
| **Attendees** | Registrant table, revenue summary | Export, Generate Google Sheet, Check In |
| **Posts** | Event announcements | + Add Post |
| **Registration** | Tickets (left) + Form builder (right) | Add Ticket, Pricing Calculator, + Add new form |
| **Vouchers** | Promo codes and discounts | Create Voucher |
| **Emails** | Email template editor | Save Template, insert variables |
| **Staff** | Team members + scanning areas | Invite Staff, New Scanning Area |

### Top Header (always visible)
- **Dashboard** — go back to event list
- **Wallet** — view earnings and withdrawals
- **Your name** (top right) — profile options

### Edit Event — 5 Sub-pages
Click **Edit Event** button (red, top right) to access:
1. **Edit Details** — event name, subdomain, dates, location, description
2. **Settings** — maintenance mode, registration options, payment settings
3. **Visual** — theme, logo, poster, background image
4. **Contacts** — organizer contact info
5. **Registration Instructions** — attendee instructions
Use the **Back** button to return to the event dashboard.

### Mobile Navigation
On mobile devices:
- Navigation tabs appear in a **hamburger menu** (☰)
- Tap hamburger → all 7 tabs are accessible
- All features work on mobile
