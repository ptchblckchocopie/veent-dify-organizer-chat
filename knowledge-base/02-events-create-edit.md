## Events — Creating, Editing & Managing

### How to Create an Event
1. Go to **Dashboard Home**
2. Click **Create Event** button
3. A dialog opens with these fields:

| Field | Required | Description |
|-------|----------|-------------|
| **Event Title** | Yes | Public name of your event |
| **Event Subdomain** | Yes | URL: your-event.veent.net |
| **Event Country / Currency** | Yes | Philippines (PHP) or Singapore (SGD). **Cannot be changed after creation.** |
| **Event Starts** | Yes | Date + time picker (Philippine Standard Time) |
| **Event Ends** | Yes | For same-day events, use same date for both |
| **Event Location** | Yes | Search field with interactive map |
| **Enable Booking Event** | No | For tours, activities, recurring sessions |
| **Event Description** | No | Rich text editor |
| **Company Logo** | No | Image upload |
| **Event Poster** | No | Image upload |
| **Background Image** | No | Image upload |

4. Click **Create Event** to finalize

### How to Edit Event Details
1. Open your event → click **Edit Event** (red button, top right)
2. You land on **Edit Details** page
3. Editable fields:
   - **Event name** — public title
   - **Subdomain** — URL slug [subdomain].veent.net (can be changed)
   - **Event Country / Currency** — READ-ONLY after creation
   - **Event address** — search field with Leaflet/OpenStreetMap
   - **Enable Recurring Event** — checkbox
   - **Event Starts / Event Ends** — date + time pickers
   - **Event description** — rich text editor
4. Click **Save changes** (changes go live immediately)

### How to Change Event Status
1. Click the **status badge** on the event header
2. Dialog opens: "Change Event Status"
3. Select new status:

| Status | Color | What Happens |
|--------|-------|-------------|
| **Live** | Green | Visible on registration page and Discover. Accepts registrations. |
| **Unlisted** | Yellow | Only accessible via direct link. Hidden from search. |
| **Unpublished** | Gray | Closed to public. Preview via bypass link only. |
| **Archived** | Red | Completely hidden. For finished/cancelled events. |

4. Click **Save Changes**

### How to End an Event
Click status badge → select **Archived** → Save Changes

### How to Duplicate an Event
From Dashboard Home event table → click three-dot menu (⋮) → select **Duplicate**

### How to Change Subdomain
Edit Event → Edit Details → change **Subdomain** field → Save changes

### Test Mode vs Live Mode
- **Test Mode**: Yellow banner shows "Payments are simulated"
- To go live: click **"Ready to go live?"** red button on the event page
- Once live, real payments are processed
