## Tickets — Creating, Editing & Pricing

### Where Are Tickets?
Click the **Registration** tab → tickets are on the **LEFT SIDE** of the page.

### How to Add a Ticket
1. Click **Registration** tab
2. Click **Add Ticket** button
3. Dialog opens with these fields:

| Field | Description |
|-------|-------------|
| **Ticket Name** | Name of the ticket type |
| **Price** | Price in ₱ (set to 0 for free tickets) |
| **Quantity** | Total stock available |
| **Ticket Image** | Optional image upload |
| **Description** | Optional description |
| **Status** | Active or Inactive |
| **Min Order** | Minimum tickets per order |
| **Max Order** | Maximum tickets per order |
| **Sales Start** | When ticket sales begin (date + time) |
| **Sales End** | When ticket sales end (date + time) |
| **Display options** | Show time for Sales Start/End |
| **Ticket Color** | Color picker for the ticket |
| **Package Ticket** | Toggle under Display Settings — bundles tickets |

4. Click **Create Ticket** to save

### How to Edit a Ticket
1. Click **Registration** tab
2. Find the ticket → click the **three-dot menu** (⋮)
3. Click **Edit Ticket**
4. Same dialog opens with current values — make changes
5. Click **Save**

### How to Delete a Ticket
1. Click **Registration** tab
2. Find the ticket → click the **three-dot menu** (⋮)
3. Click **Delete Ticket**
4. Confirm deletion

### Ticket Display Info
Each ticket card shows: name, date range, price, "Sold: X" and "Remaining: X"

### Package Tickets
Create a parent ticket with "Package Ticket" toggle enabled under Display Settings. Child tickets are linked to the parent. Parent's price is independent.

### Free Tickets
Set price to ₱0. Free orders skip payment processing.

### Ticket Status
- **Active** — visible on registration page
- **Inactive** — hidden from registration page

### What Happens When Sold Out
Tickets show "Sold Out" on registration page. You can increase stock or release reserved inventory.

### Pricing Calculator
1. Click **Registration** tab → click **Pricing Calculator** button
2. Shows current rates: Transaction Fee (7.0%), Convenience Fee (₱20)
3. Enter **Target Ticket Price**
4. Toggle **Pass on Transaction Fee** (buyer pays 7.0%)
5. Toggle **Pass on Convenience Fee** (buyer pays ₱20/ticket)
6. Enter **Number of Tickets** for total calculation
7. Fee breakdown updates in real time
