## Registration Forms — Form Builder

### Where Is the Form Builder?
Click the **Registration** tab → form builder is on the **RIGHT SIDE** of the page.

### Default Form Fields (included in every form, required)
- **First Name** * (required)
- **Last Name** * (required)
- **Contact Number** * (required)
- **Email** * (required)

These cannot be removed.

### How to Preview the Form
Click the **Preview** button above the form fields (right side).

### How to Edit Form Fields
1. Click **Registration** tab
2. On the right side, click the **Edit** button above the form fields
3. In edit mode you can:
   - **Add fields** — click **+ Add Field** button
   - **Reorder fields** — drag and drop
   - **Edit a field** — click on it to change label, type, placeholder, required
   - **Delete a field** — click delete icon on the field
4. Click **Save Changes**

### Available Field Types
| Type | Description | Example Use |
|------|-------------|-------------|
| **Text** | Single-line text | Name, company |
| **Textarea** | Multi-line text | Special requests |
| **Email** | Email with validation | Contact email |
| **Number** | Numeric input | Age, quantity |
| **Select** | Dropdown (single choice) | T-shirt size, country |
| **Multiselect** | Multiple choices | Dietary preferences |
| **Checkbox** | Yes/No | Terms agreement |
| **Date** | Date picker | Birthday |
| **File** | File upload | ID photo |

### How to Create an Additional Form
Click **+ Add new form** button at the top of the form section.

### Switching Between Forms
Multiple forms appear as **tabs** at the top. Click a tab to switch.
