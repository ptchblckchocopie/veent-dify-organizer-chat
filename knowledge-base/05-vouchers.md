## Vouchers — Promo Codes & Discounts

### Where Are Vouchers?
Click the **Vouchers** tab.

### How to Create a Voucher
1. Click **Vouchers** tab
2. Click **Create Voucher** button
3. Fill in:

| Field | Description |
|-------|-------------|
| **Promo Code** | The exact code customers type |
| **Status** | Active or Inactive |
| **Usage Type** | Single-Use, Multiple-Use, Unlimited, or Target Price |
| **Usage Limit** | Max times this code can be used |
| **Discount Type** | Percentage Off or Fixed Amount |
| **Discount Amount** | Number (e.g., 30 for 30%, or 500 for ₱500) |
| **Start Date** | Optional — leave empty for immediate |
| **Expiry Date** | Required — when voucher expires |
| **Minimum Order Amount** | Optional minimum in ₱ |
| **Applicable Ticket Types** | Which tickets this works for (or Select All) |

4. Click **Create Voucher**

### Usage Types Explained
- **Single-Use** — generates unique codes with your prefix, each works once
- **Multiple-Use** — one code, limited number of uses
- **Unlimited** — one code, no usage limit
- **Target Price** — sets a specific target price for the ticket

### How to Edit a Voucher
Click the **Vouchers** tab → find the voucher → click to edit → update fields → save

### How to Deactivate a Voucher
Change its **Status** to Inactive. It will no longer work for customers.

### Voucher Restrictions
- **Applicable Ticket Types** — select which tickets it works for
- **Minimum Order Amount** — require a minimum purchase
- **Expiry Date** — always required

### FAQ
**Q: Can vouchers make tickets free?** Yes, a 100% discount reduces price to ₱0.
**Q: Can I limit vouchers to specific tickets?** Yes, use "Applicable Ticket Types" when creating.
**Q: How do I create an invite-only event?** Enable "Require Voucher Code" in Edit Event > Settings, then create vouchers for invitees.
**Q: Why isn't my voucher working?** Check: Is it active? Is the date valid? Is there stock? Does it apply to the selected ticket?
