## Scanning & Check-in — QR Code Setup

### Setting Up Scanning Areas
1. Click **Staff** tab
2. Scroll to **Scanning Areas** section
3. Click **New Scanning Area** (red button)
4. Fill in:
   - **Name** (required) — e.g., "Main Entrance", "VIP Gate"
   - **Description** (optional)
   - **Allowed Ticket Types** — leave empty for all, or select specific tickets
   - **Settings**:
     - **Active** toggle — must be ON for check-ins
     - **Allow Multiple Scans** — allows re-entry (tickets scanned more than once)
5. Click **Create Area**

### How to Access the Scanner
From **Dashboard Home** event table → click three-dot menu (⋮) → select **Scanner**

### Using the Scanner on Event Day
1. Staff open the scanner link on their phone
2. Select the scanning area/station
3. Point camera at attendee's QR code
4. Green = valid, Red = invalid or already scanned

### Scanner Works Offline
Scans are queued locally and synced when internet connection is restored.

### Approving Scanner Devices
New devices appear in **Staff** tab → **Scanning Areas** → **For Approval** sub-tab. Manager must approve before scanning starts. If nothing pending: "All caught up! No stations pending approval."

### FAQ
**Q: Does the scanner work offline?** Yes, scans are queued and synced when connection is restored.
**Q: Can attendees check in multiple times?** By default no. Enable "Allow Multiple Scans" on the scanning area.
**Q: How do I set up check-in?** Create Scanning Areas, assign stations, approve devices, start scanning.
