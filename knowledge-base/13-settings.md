## Event Settings — Advanced Configuration

### How to Access Settings
Click **Edit Event** (red button, top right) → click **Settings** in the left sidebar.

### Section 1: Event Management
| Setting | Description |
|---------|-------------|
| **Maintenance Mode** | Shows maintenance message to users when enabled |

### Section 2: Registration
| Setting | Description | Max Length |
|---------|-------------|-----------|
| **Hide Event Description** | Hides description on registration page | — |
| **Hide QR Code on Success Page** | Hides QR after registration | — |
| **Require Invitation Code** | Requires invitation code to register | — |
| **Invite Links** | Generate shareable invite links | — |
| **Submit Button Label** | Customize submit button text | — |
| **Ticket Container Message** | Message in ticket selection area | 32 chars |
| **Ticket Selection Label** | Label for ticket dropdown | 32 chars |
| **Contact Section Heading** | Heading above contact cards | 32 chars |
| **Form Selection Label** | Label for form dropdown | 32 chars |
| **Package Ticket Label** | Label for package ticket field | 32 chars |
| **QR Code Registration Message** | Message while waiting for payment | — |

### Section 3: Payments & Tickets
| Setting | Description |
|---------|-------------|
| **Require Voucher Code** | Makes voucher mandatory (invite-only events) |
| **Max Total Tickets Per Order** | Limit tickets per order |
| **Enable Booking Tickets** | Booking mode for tours/activities |

### Saving
Click **Save Settings** at the bottom.

### Visual Sub-page
Access: Edit Event → **Visual**
- **Theme** — select theme dropdown
- **Logo** — company/organizer logo upload
- **Event Logo** — event-specific logo
- **Poster** — event poster image
- **Background Image** — registration page background
- **Embed Link** — YouTube/Vimeo URL

### Contacts Sub-page
Access: Edit Event → **Contacts**
- Click **+ Add Contact** to add organizer contact info

### Registration Instructions Sub-page
Access: Edit Event → **Registration Instructions**
- Click **+ Add Instruction** to add attendee instructions
