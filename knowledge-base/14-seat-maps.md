## Seat Maps — Reserved Seating

### Overview
Seat maps are optional. Use General Admission for unnumbered events.

### Creating a Seat Map
1. Click **Seat Map** in sidebar (if available)
2. Choose: **Create Own Map** or **Use Shared Map**
3. In the editor:
   - Click **+ Add Section** for new sections
   - Click section to add rows
   - Click row to add seats
   - Use toolbar for: zoom, pan, undo, save
4. Click **Save**

### FAQ
**Q: Can I edit a seat map after tickets are sold?** Yes, but don't remove sold seats. Adding seats is safe.
**Q: How do I share a seat map between events?** Create it and select "Shared Map" when setting up other events.
