## Quick Start Checklists — Step-by-Step Guides

### Checklist: Setting Up Your First Event
Follow these steps in order to create and launch your event:

1. **Create your event** — Dashboard Home → Create Event → fill in title, dates, location, description → Create Event
2. **Add event images** — Edit Event → Visual → upload logo, poster, background image → Save Changes
3. **Create ticket types** — Registration tab → Add Ticket → set name, price (in cents!), quantity → Create Ticket
4. **Set up registration form** — Registration tab → right side form builder → Edit → add custom fields if needed → Save Changes
5. **Configure payment settings** — Edit Event → Settings → check fee pass-on options → Save Settings
6. **Create vouchers** (optional) — Vouchers tab → Create Voucher → set code, discount, restrictions → Create Voucher
7. **Invite staff members** — Staff tab → Invite Staff → enter email, select role → Invite
8. **Customize email template** — Emails tab → edit subject, body, add variables → Save Template
9. **Test everything** — Keep event in Test Mode, try a test registration
10. **Go live** — Click "Ready to go live?" red button → then change status to **Live**

### Checklist: Day of Event (Check-in Setup)
1. **Create scanning areas** — Staff tab → Scanning Areas → New Scanning Area → name it (e.g., "Main Entrance") → Create Area
2. **Set up scanner devices** — Staff open scanner link on their phones
3. **Approve scanner devices** — Staff tab → Scanning Areas → For Approval tab → approve each device
4. **Brief staff** on check-in process — green = valid, red = invalid
5. **Have offline backup ready** — scanner works offline, scans sync when reconnected
6. **Monitor dashboard** — keep Analytics tab open to watch real-time check-ins

### Checklist: After the Event
1. **Export registrant data** — Attendees tab → Export → choose CSV, Excel, or Google Sheets
2. **Review analytics** — Analytics tab → check revenue, ticket sales, check-in rates
3. **Process any issues** — check Pending Payments in Attendees tab
4. **Request payout** — Wallet → select your event → request withdrawal
5. **Archive event** — click status badge → select Archived → Save Changes
