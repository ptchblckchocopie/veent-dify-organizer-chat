## Dashboard Home — My Events

### Overview
When you log in to hub.veent.io, you land on the Dashboard Home page. It shows a heading "Welcome back, [Your Name]" with the subtitle "Manage your events and track their performance." There is a **My Events** tab, a search bar ("Search events..."), and a **+ Create Event** button that opens the Create Event dialog.

### Event Table
Your events are listed in a table with these columns: **Event** (event name, location badge, and "Open Link" button), **Event Schedule** (date range and timezone), **Created At** (when the event was created), **Status** (dropdown to change status: Live, Unlisted, Unpublished, or Archived), **Payment** (shows "Test" or "Live" indicator), **Created By** (who created the event), and **Actions** (three-dot menu ⋮).

### Actions Menu (Three-Dot ⋮)
Click the three-dot icon on any event row to see these options: **Open Link** (opens the event's public registration page), **Scanner** (opens the scanner for this event), **Copy Link** (copies the event URL to clipboard), **Duplicate** (creates a copy of the event), and **Share** (share the event link).

### Event Status Options
You can change an event's status from the Status dropdown on the event table or from within the event management page. The four statuses are: **Live** (event is visible on registration page and Discover, attendees can register and purchase tickets), **Unlisted** (registration page is accessible via direct link only, event is hidden from Discover and search), **Unpublished** (registration is closed to the public, only accessible via bypass link for preview purposes), and **Archived** (event is completely hidden and inaccessible, use this when the event is finished or cancelled permanently).

### Create Event Dialog
Click **+ Create Event** to open the creation dialog. You need to fill in: **Event Title** (name of your event), **Event Subdomain** (URL identifier — your event will be at your-event.veent.net), **Event Country / Currency** (choose Philippines PHP or Singapore SGD — this cannot be changed after creation), **Event Starts** (date picker + time picker, Philippine Standard Time), **Event Ends** (date picker + time picker — for same-day events, use the same date for both fields), **Event Location** (search field with interactive Leaflet/OpenStreetMap map), **Event Description** (rich text editor), **Company Logo** (image upload), **Event Poster** (image upload), and **Background Image** (image upload). Click **Create Event** to finalize.
