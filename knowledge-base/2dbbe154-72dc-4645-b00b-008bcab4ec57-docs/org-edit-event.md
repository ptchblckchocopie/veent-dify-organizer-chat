## Edit Event — Overview, Details, Contacts, Instructions

Access via: **Event Management Page > Edit Event button (red pencil icon)**

The Edit Event section has a side menu with five sub-pages: Edit Details, Settings (see Settings document), Visual (see Visual document), Contacts, and Registration Instructions. There is also a Back button to return to the event management page.

### Edit Details
Heading: "Basic information". Note: "Edit your event details below. Changes update automatically on your website." The fields on this page are: **Event name** (edit the event title), **Subdomain** (edit the URL slug shown as [subdomain].veent.net), **Event Country / Currency** (read-only after creation — "Country cannot be changed after event creation"), **Event address** (search field with a Leaflet map with Clear location, Marker, and Zoom controls), **Enable Recurring Event** checkbox ("You can enable or disable recurring events at any time"), **Event Starts** (date picker + time picker in Hours:Minutes AM/PM, Philippine Standard Time), **Event Ends** (date picker + time picker — "For same-day events, use the same date for both fields"), and **Event description** (rich text editor). Buttons: Cancel and Save changes.

### Contacts
Heading: "Basic Information". Note: "Edit your contact details below. Changes update automatically on your website." Click **+ Add Contact** to add organizer contact information.

### Registration Instructions
Heading: "Registration Instructions". Note: "Add Specific Instructions for your registrants." Click **+ Add Instruction** to add step-by-step instructions for attendees.
