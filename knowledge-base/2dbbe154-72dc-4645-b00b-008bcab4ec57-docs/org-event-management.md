## Event Management Page

### Overview

When you click into an event from the Dashboard Home, you enter the Event Management page.

### Page Header

At the top you will see a **Test Mode Active** banner in yellow that says "Payments are simulated. No real transactions will be processed." (this only appears when test mode is on). There is also a **Transaction Fee** info link, a **Ready to go live?** button in red for switching from test mode to live payments, the event name, dates, location, subdomain link, creator name, an **Edit Event** button (red pencil icon) that opens the Edit Event pages, and a **Status button** showing the current event status (e.g. "Unlisted" with a yellow dot).

### Change Event Status

Click the **Status button** on the event management page to open the "Change Event Status" dialog. The dialog says "Select the visibility status for your event:" and shows four options: **Live** (event is visible on registration page and Discover, attendees can register and purchase tickets), **Unlisted** (registration page is accessible via direct link only, event is hidden from Discover and search), **Unpublished** (registration is closed to the public, only accessible via bypass link for preview purposes), and **Archived** (event is completely hidden and inaccessible, use this when the event is finished or cancelled permanently). Click **Save Changes** to apply, or **Cancel** to discard.

### Test Mode vs Live Mode

In **Test Mode**, a yellow banner is displayed and payments are simulated — no real money is processed. Use this while setting up your event. In **Live Mode**, real transactions are processed. Click "Ready to go live?" to switch from test to live.

### Seven Tabs

The event management page has seven tabs:
- **Analytics** — revenue, check-ins, ticket sales, charts, referral and voucher summaries
- **Attendees** — guest list, pending payments, export, check-in management
- **Posts** — create and publish announcements on your event page
- **Registration** — form builder, ticket types, pricing calculator
- **Vouchers** — promo codes and discount management
- **Emails** — confirmation email template editor
- **Staff** — team members, roles, and scanning areas
