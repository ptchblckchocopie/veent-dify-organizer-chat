## Edit Event — Settings Page

Access via: **Event Management Page > Edit Event button (red pencil icon) > Settings sub-page**. The Settings page has three sections and a Save Settings button at the bottom.

### Event Management & Registration Toggles
Under **Event Management** in Edit Event > Settings: **Maintenance Mode** toggle (enable/disable maintenance mode). Under **Registration** in Edit Event > Settings: **Hide Event Description** toggle (hides description on registration page, default off), **Hide QR Code on Success Page** toggle (hides QR code after registration, default off), **Require Invitation Code** toggle (requires an invitation code to register, default off), and **Generate Shareable Invite Links** toggle (under "Invite Links" heading, default off).

### Registration Page Labels & Messages
Also under **Registration** in Edit Event > Settings: **Submit Button Label** (customize submit button text), **Ticket Container Message** (message above ticket selection, max 32 chars, default "Book Your Ticket Now"), **Ticket Selection Label** (label for ticket dropdown, max 32 chars, default "Select Ticket"), **Contact Section Heading** (heading for contact section, max 32 chars, default "NEED HELP? CONTACT US"), **Form Selection Label** (label for form dropdown, max 32 chars, default "Select Form"), and **QR Code Registration Message** (message while waiting for payment QR, default "Do not close this page. After your payment is successful, your QR code will appear here within 10 seconds.").

### Payments & Tickets
Under **Payments & Tickets** in Edit Event > Settings: **Require Voucher Code** toggle (makes voucher code mandatory, default off) and **Max Total Tickets Per Order** (maximum tickets in one order, default 20).
