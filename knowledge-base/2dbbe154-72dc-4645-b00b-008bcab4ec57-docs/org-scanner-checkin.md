## Scanning Areas — Check-in Zones

Access via: **Event Management Page > Staff tab > Scanning Areas section**

The section has the heading "Scanning Areas" with the subtitle "Manage scanning areas for check-ins." There is a **New Scanning Area** button, two sub-tabs (**Areas** and **For Approval**), and a search field ("Search scanning areas..."). The Areas tab lists all created scanning areas. The For Approval tab shows scanner devices waiting for admin approval before they can be used.

### Creating a Scanning Area
Click **New Scanning Area** to open the modal. You need to fill in: **Name** (required — name of the area, e.g. "Main Entrance" or "VIP Gate"), **Description** (optional), and **Allowed Ticket Types** (leave empty to allow all ticket types, or select specific types via checkboxes). Under Settings, there is an **Active** toggle (active areas can be used for check-ins) and an **Allow Multiple Scans** toggle (allows tickets to be scanned more than once, useful for re-entry). Click **Create Area** to save, or **Cancel** to discard.

### Using the Scanner
From the Dashboard Home event table, click the three-dot menu (⋮) on an event and select **Scanner** to open the scanner for that event.
