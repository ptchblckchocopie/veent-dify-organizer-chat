## Vouchers Tab

Access via: **Event Management Page > Vouchers tab**

The Vouchers page has a heading "Vouchers", a **Create Voucher** button, and a search field ("Search voucher name") with a search button.

### Creating a Voucher
Click **Create Voucher** to open the form. You need to fill in: **Promo Code** (the exact code customers enter for multi-use vouchers), **Status** (choose Active, Inactive, or Expired), and **Usage Type** (choose one: Single-Use creates unique codes with your prefix like CODE-ABC12 where each code works once; Multiple-Use is one code with a usage limit shown in a Usage Limit field defaulting to 10; Unlimited is one code with no stock tracking needed; Target Price sets a specific final price like ₱111 for an 11.11 sale). For discounts, choose a **Discount Type** (Percentage Off or Fixed Amount) and enter the **Discount Percentage/Amount** (number input with %). For validity, set an optional **Start Date** (date + time in Philippine Standard Time — leave empty to make voucher active immediately), a required **Expiry Date** (date + time when the voucher expires), and an optional **Minimum Order Amount in ₱** (minimum cart total required). Under restrictions, use **Applicable Ticket Types** (Select All button plus individual checkboxes per ticket type) to choose which tickets this voucher applies to. Click **Create Voucher** to save, or **Cancel** to discard.
