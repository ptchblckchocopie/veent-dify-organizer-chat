## FAQ — Events, Statuses & Tickets

### Event Creation

**Q: How do I create a new event?**
A: Click the **+ Create Event** button on the Dashboard Home page. Fill in the event title, subdomain, country/currency, dates, location, and description. You can optionally upload a company logo, event poster, and background image. Click "Create Event" to finalize.

**Q: Can I duplicate an existing event?**
A: Yes. On the Dashboard Home event table, click the three-dot menu (⋮) on the event and select **Duplicate**.

**Q: How do I change my event URL/subdomain?**
A: Go to **Edit Event > Edit Details** and modify the Subdomain field. Note: changing the subdomain changes all public URLs for the event.

**Q: Can I change the event country/currency after creation?**
A: No. The event country and currency are locked after creation and cannot be changed.

### Event Statuses

**Q: What are the different event statuses?**
A:
- **Live** — Visible on registration page and Discover. Attendees can register and buy tickets.
- **Unlisted** — Registration page accessible via direct link only. Hidden from Discover and search.
- **Unpublished** — Registration closed to the public. Only accessible via bypass link for preview.
- **Archived** — Completely hidden and inaccessible. Use when event is finished or cancelled permanently.

**Q: How do I make my event visible to the public?**
A: Change the status to **Live** from the Status dropdown on the Dashboard Home event table or from the Event Management page.

**Q: How do I hide my event without deleting it?**
A: Set the status to **Archived** (completely hidden) or **Unlisted** (accessible via direct link only).

### Tickets

**Q: How do I create a ticket type?**
A: Go to the **Registration tab** in your event, then click **Add Ticket** in the Tickets section. Fill in the name, price, quantity, and other settings.

**Q: Can I have free tickets?**
A: Yes. Set the price to ₱0 when creating the ticket.

**Q: How do I set ticket sales dates?**
A: When creating or editing a ticket, use the **Sales Start** and **Sales End** date/time fields. You can optionally show the exact times to buyers.

**Q: What is a Package Ticket?**
A: Under Display Settings in the Create Ticket dialog, enable **Package Ticket** to bundle multiple tickets together as a single purchase.

**Q: How do I hide a ticket type temporarily?**
A: Toggle the **Status** off in the ticket's settings to disable it without deleting it.

**Q: How do I use the Pricing Calculator?**
A: Click the **Pricing Calculator** button in the Registration tab. Enter your target ticket price, toggle whether to pass transaction and convenience fees to the buyer, and enter the number of tickets. The calculator shows the exact fee breakdown in real time.
