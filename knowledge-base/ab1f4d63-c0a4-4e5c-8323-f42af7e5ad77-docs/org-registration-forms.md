## Registration Tab — Form Builder

Access via: **Event Management Page > Registration tab**

### Overview

- Heading: "Create Form Builder"
- Subtitle: "Manage your event form builder"
- Fee display: shows current Transaction Fee (e.g., 7.0%: Absorbed) and Convenience Fee (e.g., ₱20.00: Absorbed)
- **+ Add new form** button — create an additional registration form
- Form tabs — switch between multiple forms (e.g., "Nukie Blaze Run Registration Form")

### Default Form Fields

Every form includes these required fields by default:
- **First Name** * (required)
- **Last Name** * (required)
- **Contact Number** * (required)
- **Email** * (required)

### Form Preview & Edit

Toggle between two modes:
- **Preview** button — see how the form looks to registrants
- **Edit** button — modify form fields and layout

### Filter Tickets

Under the form, you can filter tickets by:
- **Date** (checkbox)
- **Time** (checkbox)
- **Station** (checkbox)

### Tickets Section

The Registration tab also contains the Tickets section (see the Tickets & Pricing document for details):
- **Pricing Calculator** button
- **Add Ticket** button
- Existing ticket cards showing: name, date range, price, three-dot menu, Sold count, Remaining count, progress bar
