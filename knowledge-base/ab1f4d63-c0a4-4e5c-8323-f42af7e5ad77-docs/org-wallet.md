## Wallet Page

Access via: **Top navigation > Wallet link**

### Overview

- Heading: "Wallet"
- Subtitle: "Order-based withdrawals"
- **Test Mode** toggle switch — switch between test and live mode

### Empty State

When you have no events yet, the Wallet page shows:
- "No Events Found"
- "You don't have any events yet. Create an event to start accepting orders and withdrawing funds."

### Purpose

The Wallet page is where you manage withdrawals from your event ticket sales. Withdrawals are order-based.
