## Attendees Tab — Registrant Management

### Accessing the Attendees Tab
Click the **Attendees** tab in the event navigation bar.

### What You See on This Page
The Attendees tab shows your registrant data with revenue summary and a data table.

### Revenue Summary (top)
- **Total Net Revenue** — with "View Breakdown" link and "Show/Hide" toggle
- Revenue figures are hidden by default (shown as ********) — click **Show** to reveal

### Sub-tabs
- **Registrants** — all registered attendees
- **Guest List** — guest entries
- **Pending Payments** — orders awaiting payment confirmation

### Action Buttons
- **Export** button — download attendee data (CSV, Excel)
- **Generate Google Sheet** button — export directly to Google Sheets
- **Check In (0)** — bulk check-in selected attendees
- **Undo Check In (0)** — reverse check-in for selected attendees

### Registrant Table Columns
- Checkbox (for selecting rows)
- **Ticket** — ticket type purchased
- **Name** — registrant's name
- **Phone** — contact number
- **Registered** — registration date

### Filtering
- **All Tickets** dropdown — filter by ticket type
- **Clear Filters** button — reset all filters
- **Columns** button — customize which columns are shown

### Pagination
- Shows "0 of 0 row(s) selected"
- Page navigation: Previous / Next
- Page number display (e.g., "Page 1 of 1")
