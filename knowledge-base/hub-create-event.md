## Create Event — Step by Step

### How to Create a New Event
1. Go to the **Dashboard Home** page (click "Dashboard" in the top header)
2. Click the **Create Event** button
3. A dialog/modal opens with the title "Create Event" and subtitle "Create your first event at veent"

### Create Event Form Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| **Event Title** | Text | Yes | Public title of your event |
| **Event Subdomain** | Text | Yes | URL identifier — your event will be at [your-event].veent.net |
| **Event Country / Currency** | Dropdown | Yes | Choose "Philippines (PHP)" or "Singapore (SGD)". **Cannot be changed after creation.** |
| **Event Starts** | Date + Time | Yes | Date picker + time picker. Times saved in Philippine Standard Time |
| **Event Ends** | Date + Time | Yes | For same-day events, use the same date for both fields |
| **Event Location** | Text + Map | Yes | Search field with interactive Leaflet/OpenStreetMap map |
| **Enable Booking Event** | Checkbox | No | "Attendees pick a date and time slot to book, instead of just buying a ticket. Ideal for tours, activities, hotels, and recurring sessions." |
| **Event Description** | Rich text editor | No | Detailed description of your event |
| **Company Logo** | Image upload | No | "Add logo photo" |
| **Event Poster** | Image upload | No | "Add poster photo" |
| **Background Image** | Image upload | No | "Add background photo" |

### To Finish
Click the **Create Event** button at the bottom of the dialog.

### After Creating
You'll be redirected to your new event's dashboard. The event starts in **Draft** status (not visible to public). To make it live, change the status to **Live**.
