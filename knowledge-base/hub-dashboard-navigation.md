## Dashboard Navigation — Complete Guide

### Accessing the Dashboard
1. Go to **hub.veent.io**
2. Click **Login** and sign in with your Google account
3. You land on the **Dashboard Home** page showing your events

### Dashboard Layout
The dashboard has three areas:
1. **Top Header Bar** — Veent logo, "Dashboard" link, "Wallet" link, your profile name (top right), "Edit Event" button (red)
2. **Tab Navigation** — Horizontal tabs below the event info: **Analytics**, **Attendees**, **Posts**, **Registration**, **Vouchers**, **Emails**, **Staff**
3. **Main Content Area** — Shows the content for the selected tab

### Event Info Header
At the top of every page inside an event, you'll see:
- Event name (e.g., "Nukie Blaze Run")
- Event dates (e.g., "Mar 13, 2026 → Mar 14, 2026")
- Event location
- Event public URL (e.g., https://nukierunner.veent.net)
- Organizer name
- **Edit Event** button (red, top right) — opens the Edit Event pages
- Event status badge (e.g., "Unlisted", "Live", "Draft")

### Main Navigation Tabs
Click these tabs to navigate between sections inside an event:

| Tab | What It Shows | Key Actions |
|-----|--------------|-------------|
| **Analytics** | Sales charts, revenue data, tickets by type, voucher summary, net revenue trend | View breakdown, filter by date |
| **Attendees** | Registrant table with ticket, name, phone, registration date | Export, Generate Google Sheet, Check In, Undo Check In, filter by ticket type |
| **Posts** | Event announcements/posts | Add Post button |
| **Registration** | Form builder + ticket management | Add Ticket, Pricing Calculator, Add new form, edit form fields |
| **Vouchers** | Promo codes and discounts | Create Voucher button |
| **Emails** | Email template editor | Edit template, Save Template, insert variables |
| **Staff** | Team members + scanning areas | Invite Staff, New Scanning Area |

### How to Navigate to Each Section

**To see your event analytics:**
Click the **Analytics** tab

**To see your attendee/registrant list:**
Click the **Attendees** tab

**To manage tickets:**
Click the **Registration** tab → scroll to the **Tickets** section

**To add a new ticket:**
Click the **Registration** tab → click the **Add Ticket** button

**To use the pricing calculator:**
Click the **Registration** tab → click the **Pricing Calculator** button

**To edit your registration form:**
Click the **Registration** tab → your form fields are shown on the right side (First Name, Last Name, Contact Number, Email, etc.)

**To add a new form field:**
Click the **Registration** tab → click **+ Add Field** or look for the form builder on the right

**To create a new form:**
Click the **Registration** tab → click **+ Add new form** button

**To create a voucher/promo code:**
Click the **Vouchers** tab → click **Create Voucher**

**To edit the confirmation email template:**
Click the **Emails** tab → edit the template directly on the page → click **Save Template**

**To invite a staff member:**
Click the **Staff** tab → click the **Invite Staff** button (red)

**To create a scanning area for check-in:**
Click the **Staff** tab → scroll down to **Scanning Areas** section → click **New Scanning Area** button (red)

**To edit event details:**
Click the **Edit Event** button (red, top right of the event header)

**To create a new post:**
Click the **Posts** tab → click **+ Add Post**

**To export attendee data:**
Click the **Attendees** tab → click the **Export** button or **Generate Google Sheet** button

**To go back to the dashboard home (event list):**
Click **Dashboard** in the top header bar

**To access your wallet:**
Click **Wallet** in the top header bar
