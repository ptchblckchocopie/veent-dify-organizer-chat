## Edit Details — Editing Event Information

### How to Access Edit Details
1. Open your event (click on it from Dashboard Home)
2. Click the **Edit Event** button (red button, top right)
3. You land on the **Edit Details** page (first sub-page)
4. Left sidebar shows 5 sub-pages: **Edit Details**, **Settings**, **Visual**, **Contacts**, **Registration Instructions**
5. There's a **Back** button at the top to return to the event dashboard

### Edit Details Page Layout
Heading: "Basic information"
Note: "Edit your event details below. Changes update automatically on your website."

### Editable Fields

| Field | Type | Notes |
|-------|------|-------|
| **Event name** | Text | The public title of your event |
| **Subdomain** | Text | URL slug — shown as [subdomain].veent.net. Can be changed after creation. |
| **Event Country / Currency** | Read-only | "Country cannot be changed after event creation." Shows current country/currency. |
| **Event address** | Search + Map | Search field with interactive Leaflet/OpenStreetMap map. Has +/- zoom controls. |
| **Enable Recurring Event** | Checkbox | "You can enable or disable recurring events at any time." |
| **Event Starts** | Date + Time | Date picker and time picker. Times saved in Philippine Standard Time. |
| **Event Ends** | Date + Time | "For same-day events, use the same date for both fields." |
| **Event description** | Rich text editor | Full description of your event. |

### Saving Changes
- Click **Save changes** to save your edits
- Click **Cancel** to discard changes
- Note: "Changes update automatically on your website" — once saved, they're live immediately
