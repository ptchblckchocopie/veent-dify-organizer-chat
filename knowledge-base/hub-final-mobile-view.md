## Mobile View — Dashboard on Phone/Tablet

### Mobile Navigation
On mobile devices, the dashboard adapts:
- The sidebar navigation becomes a **hamburger menu** (☰) at the top
- Tap the hamburger icon to reveal the navigation tabs
- All 7 tabs are still accessible: **Analytics, Attendees, Posts, Registration, Vouchers, Emails, Staff**
- Tap any tab to navigate — the menu closes after selection

### What's Visible on Mobile
- Event name and details header (scrollable)
- **Edit Event** button
- **Status badge** (clickable to change status)
- **Wallet** link in the top bar
- Event public URL link
- All main content for each tab

### Tips for Mobile Users
- Swipe left/right on tables to see all columns
- The form builder works on mobile but is easier on desktop
- The email template editor is fully functional on mobile
- Use pinch-to-zoom on the Leaflet map when editing event location
