## Scanner — QR Code Check-in Guide

### How to Access the Scanner
The scanner is accessed from the **Dashboard Home** event table:
1. Go to **Dashboard Home** (click "Dashboard" in the top header)
2. Find your event row in the **My Events** table
3. Click the **three-dot menu** (⋮) in the Actions column
4. Select **Scanner** from the menu

### Before Event Day — Set Up Scanning Areas
1. Go to your event → click the **Staff** tab
2. Scroll to **Scanning Areas** section
3. Click **New Scanning Area** (red button)
4. Fill in: **Name** (required, e.g., "Main Entrance"), **Description** (optional), **Allowed Ticket Types** (leave empty for all)
5. In **Settings**: toggle **Active** on, toggle **Allow Multiple Scans** if needed
6. Click **Create Area**

### On Event Day — Using the Scanner
1. Staff open the scanner link on their phone
2. Select the scanning area/station
3. Point phone camera at attendee's QR code
4. Scanner shows green for valid, red for invalid/already scanned

### Scanner Works Offline
Scans are queued locally and synced automatically when internet is restored.

### Approving Scanner Devices
New devices appear in **Staff** tab → **Scanning Areas** → **For Approval** sub-tab. Manager must approve before device can scan. If nothing pending, shows "All caught up! No stations pending approval."
