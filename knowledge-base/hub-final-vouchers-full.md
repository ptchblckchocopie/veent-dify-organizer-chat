## Vouchers — Managing Existing Vouchers

### How to View Vouchers
Click the **Vouchers** tab in the event navigation bar. You'll see a list of all created vouchers with a search bar at the top.

### Voucher List Columns
Each voucher in the list shows:
- Promo code
- Discount type and amount
- Usage type (single-use, multi-use, unlimited)
- Status (active/inactive)
- Usage count

### How to Edit a Voucher
1. Click the **Vouchers** tab
2. Find the voucher in the list
3. Click on the voucher row or its edit option
4. Update the fields as needed (code, discount, dates, applicable tickets)
5. Save changes

### How to Deactivate a Voucher
1. Click the **Vouchers** tab
2. Find the voucher
3. Change its **Status** to Inactive
4. The voucher will no longer be usable by customers

### Voucher Usage Types Explained
- **Single-Use** — generates unique codes with your prefix, each code works exactly once
- **Multiple-Use** — one code that can be used a limited number of times (set the Usage Limit)
- **Unlimited** — one code with no usage limit
- **Target Price** — sets a specific target price for the ticket

### Voucher Restrictions
- **Applicable Ticket Types** — choose which tickets the voucher works for, or select "Select All" for all tickets
- **Minimum Order Amount** — optional minimum purchase amount in ₱
- **Start Date** — when the voucher becomes active (leave empty for immediate)
- **Expiry Date** — when the voucher expires (required)
