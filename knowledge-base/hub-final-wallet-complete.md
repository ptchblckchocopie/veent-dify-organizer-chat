## Wallet — Earnings & Withdrawals

### How to Access the Wallet
Click **Wallet** in the top header bar (visible from any page).

### What You See on the Wallet Page
- Heading: "Wallet" with subtitle "Order-based withdrawals"
- **Test Mode** indicator (if event is in test mode, shows "Test Mode" — no real money)
- **Select an event** dropdown — choose which event to view earnings for
- Message: "Choose an event from the dropdown above to view your withdrawable balance and eligible orders."

### How to Use the Wallet
1. Click **Wallet** in the top header
2. Click the **Select an event** dropdown
3. Choose the event you want to check
4. You'll see your withdrawable balance and eligible orders for that event
5. Follow the prompts to request a withdrawal

### Important Notes
- Wallet shows **order-based withdrawals** — earnings are tied to specific orders
- If your event is in **Test Mode**, the wallet will show test data (no real transactions)
- You need to switch your event to **Live** payment mode before real earnings appear
- To switch from Test Mode to Live: go to your event → the Event Management page shows a "Ready to go live?" red button → click it
