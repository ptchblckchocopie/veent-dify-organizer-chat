## Registration Tab — Tickets, Forms & Pricing

### Accessing the Registration Tab
Click the **Registration** tab in the event navigation bar.

### What You See on This Page
The Registration tab has three main sections:

1. **Form Builder** (top) — heading says "Create Form Builder" with subtitle "Manage your event form builder"
2. **Fee Display** — shows current Transaction Fee (e.g., 7.0%: Absorbed) and Convenience Fee (e.g., ₱20.00: Absorbed)
3. **Tickets Section** (left side) — list of ticket types with Add Ticket button and Pricing Calculator button
4. **Form Fields** (right side) — shows the registration form fields with Preview and Edit buttons

### Ticket Management

**To add a new ticket:**
1. Click the **Registration** tab
2. Click the **Add Ticket** button in the Tickets section
3. Fill in: Ticket Name, Price, Quantity, and other settings
4. Click **Create Ticket** to save

**Existing tickets show:**
- Ticket name (e.g., "General Admission")
- Date range
- Price (e.g., ₱100)
- Three-dot menu (⋮) for more options
- Sold count and Remaining count

**To edit a ticket:**
Click the three-dot menu (⋮) next to the ticket → select Edit

### Pricing Calculator
Click the **Pricing Calculator** button in the Tickets section to see fee breakdowns.

### Form Builder

**Default form fields** (every form includes these, all required):
- **First Name** * (required)
- **Last Name** * (required)
- **Contact Number** * (required)
- **Email** * (required)

**To preview the form:**
Click the **Preview** button above the form fields

**To edit form fields:**
Click the **Edit** button above the form fields

**To add a new form:**
Click the **+ Add new form** button at the top of the form builder section

**To filter tickets:**
Use the **Filter Tickets** dropdown in the Tickets section

**Form tabs:**
If you have multiple forms, they appear as tabs (e.g., "Nukie Blaze Run Registration Form"). Click a tab to switch between forms.
