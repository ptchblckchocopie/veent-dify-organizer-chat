## Dashboard Home Page

### What You See When You Log In
After logging in to hub.veent.io, you see the Dashboard Home with:
- **Welcome message**: "Welcome back, [Your Name]" with subtitle "Manage your events and track their performance"
- **Top header bar**: Dashboard link, Wallet link, your name (top right)
- **Create Event** button
- **My Events** table showing all your events

### My Events Table Columns
| Column | Description |
|--------|-------------|
| **Event** | Event name and location, with "Open Link" to view public page |
| **Event Schedule** | Start and end dates/times |
| **Created At** | When the event was created |
| **Status** | Current status (Live, Unlisted, Draft, Ended, Archived) |
| **Payment** | Payment mode status (Live or Test) |
| **Created By** | Who created the event |
| **Actions** | Three-dot menu (⋮) with "Open menu" for more options |

### Event Actions Menu
Each event row has a three-dot menu (⋮) under the Actions column. Clicking it reveals options like:
- **Open** — opens the event dashboard
- **Scanner** — opens the scanner view for check-in
- Other event management options

### How to Open an Event
Click on the event name in the table to enter that event's dashboard.

### How to Create a New Event
Click the **Create Event** button at the top right of the My Events section.
