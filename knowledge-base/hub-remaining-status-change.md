## Event Status — How to Change and What Each Status Means

### How to Change Event Status
1. Open your event (click on it from the Dashboard Home)
2. Look for the **status badge** near the top of the page (it shows the current status like "Unlisted", "Draft", etc.)
3. Click on the status badge
4. A dialog opens titled **"Change Event Status"** with the text "Select the visibility status for your event"
5. Select the new status you want
6. Click **Save Changes**

### Status Options

| Status | Color | Description |
|--------|-------|-------------|
| **Live** | Green | Event is visible on registration page and Discover. Attendees can register and purchase tickets. |
| **Unlisted** | Yellow | Registration page is accessible via direct link only. Event is hidden from Discover and search. |
| **Unpublished** | Gray | Registration is closed to the public. Only accessible via bypass link for preview purposes. |
| **Archived** | Red | Event is completely hidden and inaccessible. Use this when the event is finished or cancelled permanently. |

### Common Questions

**"How do I make my event live?"**
Click the status badge → select **Live** → click **Save Changes**

**"How do I hide my event?"**
Click the status badge → select **Unlisted** (still accessible via link) or **Unpublished** (fully closed) → click **Save Changes**

**"How do I end/finish my event?"**
Click the status badge → select **Archived** → click **Save Changes**

**"How do I unpublish my event?"**
Click the status badge → select **Unpublished** → click **Save Changes**
