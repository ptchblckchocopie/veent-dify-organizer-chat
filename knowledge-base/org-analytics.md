## Analytics & Reporting

### Dashboard Analytics

Access via: **Event Page > Analytics**

#### Key Metrics
| Metric | Description |
|--------|-------------|
| **Total Revenue** | Gross sales amount |
| **Net Revenue** | Revenue after fees |
| **Tickets Sold** | Total tickets sold |
| **Orders** | Total order count |
| **Check-ins** | Attendees checked in |

#### Time Period Filters
- Last 7 days
- Last 30 days
- Custom date range
- All time

#### Currency Filter
- PHP (Philippines)
- SGD (Singapore)
- USD (US Dollar)

### Financial Summary

#### Ticket Summary Grid
| Column | Description |
|--------|-------------|
| **Ticket Type** | Ticket name |
| **Qty Sold** | Number sold |
| **Gross Revenue** | Total before fees |
| **Net Revenue** | After fees |
| **Avg Price** | Average ticket price |

#### Voucher Summary
| Column | Description |
|--------|-------------|
| **Voucher Code** | Promo code |
| **Uses** | Times used |
| **Total Discount** | Total discount given |

### Data Export

#### Export Formats
| Format | Description |
|--------|-------------|
| **CSV** | Spreadsheet-compatible |
| **Excel** | Microsoft Excel format |
| **Google Sheets** | Direct export to Sheets |

#### Exportable Data
- Registrant list
- Order details
- Check-in log
- Financial summary
