## Email & Communications

### Email Templates

Access via: **Event Page > Emails**

#### Template Fields
| Field | Description |
|-------|-------------|
| **Title** | Email subject line |
| **Email From** | Sender address |
| **Email To** | Recipient (use `{{email}}`) |
| **CC/BCC** | Carbon copy recipients |
| **Content** | Email body (HTML) |

#### Available Variables
| Variable | Description |
|----------|-------------|
| `{{firstName}}` | Registrant's first name |
| `{{lastName}}` | Registrant's last name |
| `{{email}}` | Registrant's email |
| `{{contactNumber}}` | Phone number |
| `{{tickets}}` | Ticket card HTML |
| `{{inviteUrls}}` | Private invite links |

### Event Announcements

Access via: **Event Page > Posts**

#### Creating Posts
1. Click "Add Post"
2. Enter title and content (rich text editor)
3. Add image (optional)
4. Set status: Draft or Published
5. Save

#### Post Fields
| Field | Description |
|-------|-------------|
| **Title** | Post heading |
| **Content** | Post body (HTML) |
| **Image** | Featured image |
| **Status** | draft/published |
| **Publish Date** | When to publish |

### Subscription Management
- Users can unsubscribe from event emails
- One-click unsubscribe links in all emails
- Manage subscriptions via admin panel
