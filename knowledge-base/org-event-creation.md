## Event Creation & Basic Settings

### Creating a New Event

To create a new event, click the "Create Event" button on the dashboard home page.

#### Required Fields
| Field | Description | Example |
|-------|-------------|---------|
| **Event Title** | Name of your event | "Summer Music Festival 2025" |
| **Subdomain/Slug** | Unique URL identifier (auto-generated from title) | "summer-music-festival-2025" |
| **Start Date & Time** | When the event begins | "2025-06-15 at 6:00 PM" |
| **End Date & Time** | When the event ends | "2025-06-15 at 11:00 PM" |
| **Location** | Venue address or virtual location | "SM Mall of Asia Arena" |
| **Description** | Event details (supports HTML) | Full event description |

#### Optional Fields
| Field | Description |
|-------|-------------|
| **Event Logo** | Logo image for the event |
| **Company Logo** | Organizer's company logo |
| **Poster Image** | Main promotional image |
| **Background Image** | Background for registration page |
| **Embed Link** | YouTube/Vimeo embed URL |
| **Event Country** | Country where event takes place |
| **Category** | Event category for discovery |
| **Timezone** | Event's local timezone |

### Event Status Options
| Status | Description |
|--------|-------------|
| **Draft** | Event is being set up, not visible to public |
| **Live** | Event is active and accepting registrations |
| **Ended** | Event has concluded |

### Event Settings (Advanced)

Access via: **Event Page > Edit > Settings**

| Setting | Description | Default |
|---------|-------------|---------|
| **Ticket Purchase Type** | "multiple" (concert-style) or "single" (fun run-style) | multiple |
| **Payment Active** | Enable/disable payment processing | true |
| **Recurring Event** | Mark as recurring event | false |
| **Event Finished** | Mark event as completed | false |
| **Disable Registration** | Stop accepting new registrations | false |
| **Maintenance Mode** | Show maintenance message to users | false |
| **Bulk Registration** | Allow multiple tickets per order | true |
| **Bulk Limit** | Maximum tickets per order | 10 |
| **Max Total Tickets Per Order** | Hard limit on total tickets | 10 |
| **Reservation Expiry (Minutes)** | Time before unpaid reservations expire | 15 |

### Customization Labels
| Setting | Description | Default |
|---------|-------------|---------|
| **Register Button Label** | Text on main registration button | "Register" |
| **Submit Button Label** | Text on form submit button | "Submit" |
| **Ticket Container Message** | Message above ticket selection | "" |
| **Select Ticket Label** | Label for ticket selection | "Select Ticket" |
| **Contact Us Message** | Support contact information | "" |
| **Confirmation Message** | Message shown after registration | "" |

### Visibility Settings
| Setting | Description | Default |
|---------|-------------|---------|
| **Hide Event Description** | Hide description on registration page | false |
| **Hide Tickets on Success Page** | Don't show tickets on confirmation | false |
| **Hide Voucher Input** | Hide promo code input field | false |
| **Require Voucher Code** | Make voucher required (invite-only) | false |
