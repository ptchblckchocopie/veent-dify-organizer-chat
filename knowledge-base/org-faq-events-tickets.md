## FAQ — Event Setup, Tickets & Seat Maps

### Event Setup

**Q: How do I create a new event?**
A: Click "Create Event" on the dashboard home page. Fill in the event title, dates, location, and description. You can add tickets and forms after creating the event.

**Q: Can I duplicate an existing event?**
A: Currently, events must be created individually. However, you can use shared seat maps and copy form configurations manually.

**Q: How do I change my event URL/subdomain?**
A: Edit the event and modify the subdomain field under Details. Note: Changing the subdomain will change all public URLs.

**Q: What's the difference between Draft and Live status?**
A: Draft events are not visible to the public. Live events appear on the registration page and accept purchases.

**Q: How do I end my event?**
A: Go to Settings and enable "Event Finished". This stops new registrations but keeps data accessible.

### Tickets & Pricing

**Q: How do I set ticket prices?**
A: Prices are entered in cents. For example, ₱1,000.00 = 100000 cents.

**Q: Can I have free tickets?**
A: Yes, set the price to 0. Free orders skip payment processing.

**Q: How do package tickets work?**
A: Create a parent ticket with "Is Package" enabled, then create child tickets linked to it. The parent's price is independent of children.

**Q: What happens when tickets sell out?**
A: Tickets show as "Sold Out" on the registration page. You can increase stock or release reserved inventory.

**Q: How do I hide a ticket type?**
A: Set the ticket status to "Inactive". It won't appear on the registration page.

### Seat Maps

**Q: Do I need a seat map for my event?**
A: No, seat maps are optional. Use General Admission for unnumbered events.

**Q: Can I edit a seat map after tickets are sold?**
A: Yes, but be careful not to remove sold seats. Adding seats is always safe.

**Q: How do I share a seat map between events?**
A: Create a seat map and use "Shared Map" when setting up other events.
