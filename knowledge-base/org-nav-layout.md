## Dashboard Navigation — Layout & Getting Started

### Dashboard Layout

The dashboard has three main areas:
1. **Top Header** - Logo, user profile, global navigation (Wallet, Orders)
2. **Left Sidebar** - Event-specific navigation tabs (appears after selecting an event)
3. **Main Content Area** - The current page content

### Step-by-Step: Getting Started

#### Accessing the Dashboard
1. Go to `dashboard.veent.io`
2. Click **"Sign In"** (top right)
3. Sign in with your Clerk account (email/Google/Facebook)
4. You'll land on the **Home page** showing your events

#### Creating Your First Event
1. From **Home**, click the **"+ Create Event"** button (top right)
2. Fill in the form:
   - Event Title (required)
   - Start/End Date & Time (required)
   - Location (required)
   - Description
3. Click **"Create Event"**
4. You'll be redirected to your new event's dashboard

### Main Navigation Tabs

After selecting an event, the **left sidebar** shows these tabs:

| Tab | What You'll Find | Common Actions |
|-----|------------------|----------------|
| **Overview** | Event dashboard summary | View quick stats, recent activity |
| **Analytics** | Sales charts, revenue data | View sales trends, export reports |
| **Registrants** | Attendee list, data table | Search, filter, export, resend emails |
| **Form** | Registration form builder | Add fields, reorder, set required |
| **Seat Map** | Venue configuration | Create/edit sections, rows, seats |
| **Emails** | Email templates | Edit confirmation email, preview |
| **Posts** | Announcements | Create/edit event announcements |
| **Staff** | Team management | Invite staff, assign roles |
| **Promotion** | Vouchers & discounts | Create vouchers, view usage |
| **Scanning Areas** | Check-in zones | Create areas, manage stations |
| **Printed Tickets** | Physical tickets | Allocate, distribute, void |
| **Edit** | Event settings | Opens edit submenu |

### Edit Event Submenu

Click **"Edit"** in the sidebar to see these sub-pages:

| Sub-page | What to Configure | How to Access |
|----------|-------------------|---------------|
| **Details** | Title, dates, location, description, subdomain | Edit > Details |
| **Contacts** | Organizer contact info, support email | Edit > Contacts |
| **Instructions** | Attendee instructions, entry requirements | Edit > Instructions |
| **Visuals** | Logo, poster, background images | Edit > Visuals |
| **Settings** | Advanced settings (fees, limits, visibility) | Edit > Settings |
