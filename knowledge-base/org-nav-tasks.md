## Dashboard Navigation — Common Tasks & Quick Reference

### How to Access Common Tasks

#### Creating Tickets
1. From your event, click **"Edit"** > **"Details"** in sidebar
2. Scroll to **"Ticket Types"** section
3. Click **"+ Add Ticket Type"**
4. Fill in: Name, Price (in cents), Stock, Description
5. Click **"Save"** at bottom of page

#### Setting Up Registration Form
1. Click **"Form"** in sidebar
2. Click **"+ Add Field"** button
3. Choose field type (Text, Email, Dropdown, etc.)
4. Configure: Label, Required, Placeholder
5. Drag fields to reorder
6. Click **"Save Changes"**

#### Creating a Voucher/Promo Code
1. Click **"Promotion"** in sidebar
2. Click **"+ Create Voucher"** button
3. Fill in:
   - Prefix (e.g., "EARLYBIRD")
   - Discount Type (Percentage or Fixed)
   - Amount (e.g., 20 for 20% or 50000 for ₱500)
   - Usage Type (Single, Multiple, Unlimited)
   - Stock (for limited use)
4. Click **"Create"**

#### Inviting Staff Members
1. Click **"Staff"** in sidebar
2. Click **"+ Invite Staff"** button
3. Enter their email address
4. Select role: Manager, Editor, Scanner, or Viewer
5. Click **"Send Invite"**
6. They'll receive an email to join

#### Setting Up Check-in Scanning
1. Click **"Scanning Areas"** in sidebar
2. Click **"+ Create Area"** to add a check-in zone
3. Name the area (e.g., "Main Entrance", "VIP Gate")
4. Add stations by clicking **"+ Add Station"**
5. On event day, open `/scanner` on your phone
6. Select your station and start scanning

#### Configuring Seat Map
1. Click **"Seat Map"** in sidebar
2. Choose: **"Create Own Map"** or **"Use Shared Map"**
3. In the editor:
   - Click **"+ Add Section"** for new sections
   - Click section to add rows
   - Click row to add seats
   - Use toolbar for: zoom, pan, undo, save
4. Click **"Save"** when done

#### Publishing Your Event (Go Live)
1. Click **"Edit"** > **"Settings"**
2. Find **"Event Status"** section
3. Change status from **"Draft"** to **"Live"**
4. Confirm in the Go Live modal
5. Your event is now public at `[subdomain].veent.io`

### Global Navigation (Top Header)

Click your **profile icon** (top right) to access:

| Menu Item | What It Does |
|-----------|--------------|
| **Wallet** | View earnings, request withdrawals |
| **Orders** | See all orders across events (admin) |
| **Withdrawals** | Manage payout requests |
| **Settings** | Account preferences |
| **Sign Out** | Log out of dashboard |

### Quick Access Buttons

| Page | Primary Button | What It Does |
|------|----------------|--------------|
| Home | "+ Create Event" | Start new event |
| Registrants | "Export" | Download attendee data |
| Registrants | "Resend Email" (per row) | Resend confirmation |
| Form | "+ Add Field" | Add form field |
| Promotion | "+ Create Voucher" | New promo code |
| Staff | "+ Invite Staff" | Add team member |
| Scanning Areas | "+ Create Area" | New check-in zone |
| Printed Tickets | "Allocate" | Assign printed tickets |

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + S` | Save changes (in editors) |
| `Escape` | Close modal/dialog |
| `Enter` | Confirm action in dialogs |

### Mobile Navigation

On mobile devices:
1. The sidebar collapses into a **hamburger menu** at top left
2. Tap the menu icon to reveal navigation tabs
3. Tap any tab to navigate
4. The menu auto-closes after selection
