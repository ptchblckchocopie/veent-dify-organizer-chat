## Registration Forms

### Form Builder

Access via: **Event Page > Form**

#### Creating Forms
1. Click "Create Form" button
2. Enter form title and description
3. Add custom fields using the form builder
4. Associate ticket types with the form

#### Field Types Available
| Type | Description | Use Case |
|------|-------------|----------|
| **text** | Single-line text input | Name, phone number |
| **textarea** | Multi-line text | Special requests |
| **email** | Email input with validation | Contact email |
| **number** | Numeric input | Age, quantity |
| **select** | Dropdown selection | T-shirt size, country |
| **multiselect** | Multiple selection | Dietary preferences |
| **checkbox** | Yes/No checkbox | Terms agreement |
| **date** | Date picker | Birthday |
| **file** | File upload | ID photo |

#### Field Properties
| Property | Description |
|----------|-------------|
| **Label** | Display name for the field |
| **Name** | Internal field identifier (no spaces) |
| **Required** | Make field mandatory |
| **Description** | Help text below field |
| **Options** | Choices for select/multiselect |
| **Placeholder** | Example text in input |

### Form-Ticket Association
- Each form can be linked to multiple ticket types
- Registrants see the form associated with their selected ticket
- Different tickets can have different forms
