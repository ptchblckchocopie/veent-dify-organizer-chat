## Seat Maps & Venue Setup

### Seat Map Editor

Access via: **Event Page > Seat Map**

#### Creating a Seat Map
1. Choose "Create Own Map" or "Use Shared Map"
2. Upload venue floor plan (optional)
3. Add sections and rows
4. Configure seat numbering

#### Section Types
| Type | Description |
|------|-------------|
| **Reserved Seating** | Individual numbered seats |
| **General Admission (GA)** | Unnumbered area with capacity |

#### Section Properties
| Property | Description |
|----------|-------------|
| **Name** | Section identifier (e.g., "Orchestra A") |
| **Color** | Visual color on map |
| **Ticket Type** | Associated ticket type |
| **Capacity** | Total seats/spots |
| **Status** | active/inactive |

#### Row Configuration
| Property | Description |
|----------|-------------|
| **Row Label** | Row identifier (A, B, C or 1, 2, 3) |
| **Seat Count** | Number of seats in row |
| **Seat Numbering** | Start number and direction |
| **Curve** | Row curvature for arced sections |

#### Seat Properties
| Property | Description |
|----------|-------------|
| **Seat Label** | Seat identifier |
| **Status** | available/disabled/blocked |
| **Accessibility** | Mark as accessible seating |

### Transform Tools
| Tool | Function |
|------|----------|
| **Rotation** | Rotate section/row |
| **Curve** | Add arc to row |
| **Stretch X/Y** | Scale horizontally/vertically |
| **Flip H/V** | Mirror section |

### 3D Preview
- Toggle between 2D editor and 3D preview
- View venue from attendee perspective
- Isometric and top-down views

### Sharing Seat Maps
- Seat maps can be shared between events
- Shared maps update across all linked events
- Use "own-map" for event-specific customizations
