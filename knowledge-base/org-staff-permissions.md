## Staff & Permissions

### Team Roles

Access via: **Event Page > Staff**

#### Role Types
| Role | Description |
|------|-------------|
| **Manager** | Full access to all event features |
| **Editor** | Edit content, manage tickets/vouchers/forms |
| **Scanner** | Check-in only |
| **Viewer** | Read-only access |

### Permission Matrix

| Permission | Manager | Editor | Scanner | Viewer |
|------------|---------|--------|---------|--------|
| Edit Event | Yes | Yes | No | No |
| Manage Staff | Yes | No | No | No |
| View Settings | Yes | No | No | No |
| Create Posts | Yes | Yes | No | No |
| Check-In | Yes | Yes | Yes | No |
| Download Data | Yes | Yes | No | No |
| Manage Tickets | Yes | Yes | No | No |
| Manage Vouchers | Yes | Yes | No | No |
| Send Emails | Yes | No | No | No |

### Inviting Staff
1. Navigate to Staff page
2. Click "Invite Staff"
3. Enter email address (must have Veent account)
4. Select role (Manager, Editor, Viewer)
5. Click Submit

### Scanning Areas

Access via: **Event Page > Scanning Areas**

| Field | Description |
|-------|-------------|
| **Name** | Area identifier |
| **Allowed Ticket Types** | Which tickets can scan here |
| **Multiple Scan** | Allow re-scanning same ticket |
| **Stations** | Linked scanner devices |
